IMPORTANT: This chart requires a pre-existing PVC for the data volume.
The release was installed with:

  persistence.data.existingClaim: {{ .Values.persistence.data.existingClaim }}

That PVC must exist in namespace {{ .Release.Namespace }} before install, or the pod will fail to start.
To change it, set --set persistence.data.existingClaim=<your-pvc-name> at install time.
